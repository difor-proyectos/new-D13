# DIFOR brand spec

Fuentes usadas:
- Paginas descargadas desde `https://www.difor.cl/`, `maxus-chile`, `jetour-chile`, `karry-chile` y `nuestros-servicios`.
- Imagenes de referencia adjuntas del proyecto.

## Tokens base

```css
:root {
  --bg: oklch(20.5% 0.020 248.8);
  --surface: oklch(22.0% 0.012 254.1);
```

## Tipografia

- Display: `"BMWTypeNext", "Helvetica Neue", Arial, sans-serif`
- Body: `"BMWTypeNext", "Helvetica Neue", Arial, sans-serif`
- Mono: `"SF Mono", ui-monospace, Menlo, monospace`

## Postura de layout

- Fondo negro/azul profundo con superficies oscuras apenas elevadas.
- Naranjo DIFOR como accion principal, foco de estado y contorno puntual; no usarlo como lavado de fondo.
- Bordes finos y rectos, radio minimo, tarjetas sobrias con alta legibilidad.
- Fotografia automotriz protagonista con overlay oscuro para proteger contraste.
- Inicio enfocado en venta: tres accesos destacados solamente, navegacion secundaria disponible pero no protagonista.
